fn progress_pulse(&self)[src]

Indicates that some progress is made, but you don’t know how much.

Causes the entry’s progress indicator to enter “activity mode”, where a block bounces back and forth. Each call to Self::progress_pulse() causes the block to move by a little bit (the amount of movement per pulse is determined by [methodGtk.Entry.set_progress_pulse_step]).
